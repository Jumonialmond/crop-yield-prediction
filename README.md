# Crop Yield Prediction

Exploratory data analysis and machine learning model to predict crop yield based on environmental and agricultural factors.

**Status:** Work in progress

**Tools:** Python, pandas, matplotlib, seaborn, scikit-learn, MLflow

**Data source:** Agriculture Crop Yield Dataset (Kaggle)

## Key Questions
- Which factors most influence crop yield?
- How do rainfall, temperature, and soil type interact to affect productivity?
- Can we build a reliable predictive model for yield estimation?
